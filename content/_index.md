Welcome to this hands-on guide for setting up a modern [Hugo](https://gohugo.io) website. Hugo is one of the fastest static site generators, written in Go.

In the following steps you will learn how to install Hugo using [mise](https://mise.jdx.dev), create a new project, add a theme, write content, and deploy your site to GitHub Pages.

Work through the guide below in order to get your first Hugo site up and running.
